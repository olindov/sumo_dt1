# Deployment Notes

## Suggested pilot sites in Sarajevo / East Sarajevo
- major signalized intersections
- school safety zones
- corridor segments with congestion and air-quality concerns
- public transport priority corridors

## Suggested KPI set
- average speed
- travel time
- average queue length
- CO2 emissions
- PM / NOx where emission model is enabled
- number of stops
- junction throughput

## Expansion path
1. Single-camera counting pilot
2. Single-junction DT
3. Corridor model
4. Multi-camera integrated Living Lab
5. City dashboard and cross-scenario testing
